UX Aspects (AngularJS) end of life

December 31st 2021

As AngularJS 1.x reaches end of life at the end of this year, we have now retired the AngularJS version of UX Aspects. The UX Aspects team will make no further releases of the AngularJS component library.


What should I do if I need an AngularJS bug fix or feature?

Assuming you are unable to use the Angular components, the only option will be to clone the repository and make the change yourself. You will also need to take responsibility for the ITLS declaration of any future releases involving UX Aspects (AngularJS).


How can I tell if a project is using the affected library?

  • Run the following command: npm ls @ux-aspects/ux-aspects-ng1
  • If it shows (empty) then this product retirement does not affect your application.
