UX Aspects (AngularJS) end of life

December 31st 2021

Because AngularJS 1.x has reached the end of its support window, we have now retired the AngularJS version of UX Aspects. The UX Aspects team will make no further releases of the AngularJS component library.

What should I do if I need an AngularJS bug fix or feature?

The recommended approach is to use the UX Aspects Angular components instead. If you are unable to migrate, the only option will be to clone the repository and make the change yourself. You will also need to take responsibility for the legal declaration of any future releases involving UX Aspects (AngularJS).

How can I tell if a project is using the affected library?

Run the following command in the root of your project:

npm ls @ux-aspects/ux-aspects-ng1

If it shows (empty) then this product retirement does not affect your application.
